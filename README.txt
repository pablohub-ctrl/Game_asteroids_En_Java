Juego asteroids


Caso 1: 		-valor para N iguales a 5 o hasta 12, utilizar un rango de [5-12]
			-valor para M iguales a 10 o hasta 12, utilizar para M un rango de [10-12]
			-valor para A iguales a 2 o hasta 3, utilizar para A un rango de [2-3]
			-valor para L (fireRange) iguales a 2 o hasta menor a (N-2). Ejemplo: si N es igual a 5 entonces el 			rango máximo para L es [2] o si N es igual a 7 entonces el rango máximo para L es de [2-4].
			-valor para seed será siempre cualquier valor mayor a cero.


 Se puede ver que cada partida te indica que opción utilizar:
1.- Actualizar partida: se refrescan los elementos del espacio en un tiempo + 1.
2.- Lanzar un proyectil: la nave lanza un proyectil con el sentido de la misma dirección a la nave.
3.- Efectuar movimiento: efectúa una actualización de la posición de la nave, la cual se mantiene en movimiento en el espacio por dos espacios más después de efectuar el movimiento.
Las partidas para caso inician con una cantidad de asteroides definida por el usuario con respecto a los valores en rango y por cada actualización se integrará la misma cantidad hasta cumplir un límite definido por cada caso, una vez superado ese límite cada actualización no se incorporarán más asteroides. 

Las opciones presentes para efectuar un movimiento son de: 0 grados, 45, 90, 135, 180, 225 y 270 grados.
